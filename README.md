-   **Premium Design**: Glassmorphism UI with smooth animations.
-   **Client-Side Processing**: All conversions happen in your browser for privacy.

## Tech Stack

-   Vite
-   Vanilla JavaScript
-   CSS3 (Glassmorphism)
-   jsPDF

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open `http://localhost:5173` in your browser.

## Build

To build for production:

```bash
npm run build
```
