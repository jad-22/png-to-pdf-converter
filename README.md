# PNG to PDF Converter

A premium, modern web application to convert PNG images to PDF files.

## Features

-   **Convert PNG to PDF**: Instantly convert your images.
-   **Batch Processing**: Combine multiple PNGs into a single PDF file.
-   **Drag & Drop**: Easy to use interface.
-   **Premium Design**: Glassmorphism UI with smooth animations.
-   **Client-Side Processing**: All conversions happen in your browser for privacy.

## Tech Stack

-   Vite
-   Vanilla JavaScript
-   CSS3 (Glassmorphism)
-   jsPDF

## Setup

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open `http://localhost:5173` in your browser.

## Build

To build for production:

```bash
npm run build
```
